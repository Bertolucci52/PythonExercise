C'è un video che illustra i passi elencati in questo
documento.

Installare python dopo averlo scaricato 
dal sito www.python.org.

Attenzione, nella procedura di installazione inserire python 
nella variabile d'ambiente PATH in modo che sia raggiungibile
da ogni punto. 

Creare una directory di lavoro. 

Copiare in tale directory il materiale di questa Etivity. 

Digitare il comando 

python --version 

da PROMPT dei comandi (TERMINALE per utenti MAC, CMD per utenti WINDOWS)

Eseguire tutti gli step indicati
nel file TutorialPython.py

Al termine, eseguire Esercizio.txt .
